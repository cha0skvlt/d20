# d20 Bot

A simple Telegram bot for rolling dice expressions.

## Setup

1. Copy `.env.example` to `.env` and add your `TELEGRAM_BOT_TOKEN`.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the bot:
   ```bash
   python -m bot.main
   ```

## Testing

Run `pytest` to execute tests.
